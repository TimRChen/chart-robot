# intelligent-assistant-robot
A Robot use Tensorflow.js

### 0.0.1 alpha

-   初始化机器人UI及输入输出界面
-   支持问答及简单回复

### How to import
>   注意，当前暂时无法这样引用
```js
import cqrobot from 'intelligent-assistant-robot';
cqrobot(); // that's done
```

### How to build
```bash
$ npm start
```
```bash
$ npm run dev
```
